# IntelliProctor
IntelliProctor is a smart AI-powered exam monitoring system designed to ensure fair and secure online examinations. The system uses computer vision techniques to monitor candidate behavior in real-time through a webcam. It detects suspicious activities such as looking away from the screen, multiple faces, or absence from the frame.

🛠️ Technologies Used
Frontend:
HTML → Structure of the web pages
CSS → Styling and UI design
Backend:
Python → Core logic and AI processing
Flask → Web framework to connect frontend and backend
Computer Vision:
OpenCV (CVE you mentioned probably means OpenCV 😉) → Face detection and tracking


⚙️ Key Features 
🎥 Real-time face detection
👁️ Eye movement tracking
🚨 Suspicious activity alerts
📝 Logging system (CSV / database)
🌐 Web-based exam interface
⏱️ Continuous monitoring during exam
